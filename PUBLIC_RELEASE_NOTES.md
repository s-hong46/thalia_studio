# Public release notes

This repository snapshot was prepared for public GitHub upload.

## Removed from the public snapshot

- `.env` and local credentials
- one off helper scripts that contained hard coded secrets or local scratch logic
- generated rehearsal recordings, preview clips, and TTS files
- local caches, pytest temp folders, and runtime artifacts
- local test databases and journals

## Kept in the public snapshot

- application source code
- tests
- setup scripts
- documentation and repository policy files
